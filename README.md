# Pure HTML Projects

This repository contains various projects developed using pure HTML. Below is a brief description of each HTML file found in this repository:

## Projects

### contact_form.html
A simple contact form that allows users to send their inquiries. Includes fields for name, email, subject, and message.

### homepage.html
A sleek homepage template that serves as the main entry point for showcasing the website, featuring navigation links and an introduction.

### links.html
A page that contains useful external links, categorized for easy access. It's a great resource for references and additional information.

### portfolio.html
A professionally designed portfolio page that highlights projects and achievements. Suitable for freelancers and professionals to showcase their work.

### recipes.html
A collection of HTML pages that display various recipes, including ingredients and detailed instructions.

### resume.html
An HTML-based resume template that presents work experience, education, and skills in a structured format.

### tables.html
A page that demonstrates how to create and style tables in HTML, showcasing various data representations.
