## 📌 Key Concepts by Level

### ✅ Level 1 (Beginner)
- Basic HTML structure (`<!DOCTYPE>`, `<html>`, `<head>`, `<body>`)
- Semantic elements: `<header>`, `<footer>`, `<section>`
- Text content: `<h1>-<h6>`, `<p>`, `<ul>`, `<li>`
- Links: `<a>` with `href` attribute

### ✅ Level 2 (Intermediate)
- Image embedding: `<img>` with `src`, `alt`, `width` attributes
- Table basics: `<table>`, `<tr>`, `<td>`, `<th>`
- Form elements: `<form>`, `<input>`, `<label>`, `<textarea>`
- Semantic layout: `<header>`, `<main>`, `<section>`, `<footer>`

### ✅ Level 3 (Advanced)
- Complex tables: `<thead>`, `<tbody>`, `<caption>`, `colspan`
- Form input types: radio buttons, checkboxes, submit buttons
- SVG graphics: shapes, attributes, styling
- Accessibility considerations

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/a-man77/pure_html_projects.git
   cd pure_html_projects
   ```

2. **Open any HTML file in your browser:**
   - Right-click the file → "Open with" → Choose your browser
   - Or drag and drop the file into your browser window

3. **Edit and Experiment:**
   - Open files in a text editor (VS Code, Sublime Text, etc.)
   - Modify the HTML and refresh your browser to see changes
   - Try creating your own variations!

## 💡 Tips for Learning

- 📖 Read the HTML source code line by line
- 🎨 Modify colors, text, and structure to experiment
- 🔍 Inspect elements (Right-click → Inspect) to understand structure
- 📝 Comment out sections to see what changes
- 🎯 Try combining concepts from different files
